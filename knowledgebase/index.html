What are the important details about the setup of this QA tenant related to service levels?
Question:
When I try to create a contact from the Customer Info page, I get the error message "Access Denied". Why can't I add a new contact?

Answer:
The service level configuration within this QA site is as follows.
Service Levels:
* The default time for all topics in Segment1 and Segment2 is 60 minutes.
* Topic S1Topic Int Prop Svc Lvl - 20 min has the interaction property of "20 Minute Interaction"
associated with it.
This property sets a service level of 20 min for any question in that topic that has the property
value set to True.
* Topic S1 SvcLvl10-min has a 10 minute service level.
* Customer5@1to1service.com has a 5 minutes service level.
* Topic S1TopicSvcLvl1-min as a 1 minute service level

Order of Testing:
1- S1topic1 (60 min SL)
2- S1Topic-Int Prop Svc Lvl-20
3- S1TopicSvcLvl-10
4- S1Topic1 but submitted from customer5
5- S1TopicSvcLvl-1

These messages should then be delivered in reverse order to the agent - 5, 4, 3, 2, then 1.
